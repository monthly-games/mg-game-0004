## 3. 상세 BM 설계

### 3.1 스태미너/라이프 BM

#### 무료 구조
- 라이프 최대 5개, 30분당 1 회복
- 일일 무료 충전 1회 (광고)
- 퀘스트 완료 시 라이프 보상

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 라이프 즉시 충전 | 5개 회복 | 프리미엄 재화 100 |
| 무한 라이프 | 2시간 동안 무제한 | $1.99 |
| 라이프 패스 | 최대 +3, 회복 15분 (30일) | $4.99/월 |
| 광고로 라이프 | 1개 회복 | 무료 (광고) |

### 3.2 꾸미기/인테리어 BM

#### 무료 구조
- 스테이지 클리어로 기본 인테리어 획득
- 스토리 진행으로 카페 확장
- 이벤트에서 한정 데코 파밍 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 인테리어 세트 (가구 15종+) | $4.99~$9.99 |
| 단품 장식 | 개별 프리미엄 장식 | $0.99~$2.99 |
| 시즌 한정 세트 | 시즌별 테마 데코 | $6.99 |
| 메뉴 스킨 | 음식/음료 외형 변경 | $1.99 |

### 3.3 부스터/편의 BM

#### 무료 구조
- 스테이지 클리어로 기본 부스터 획득
- 일일 미션에서 부스터 보상
- 이벤트에서 추가 부스터 지급

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 부스터 팩 | 각종 부스터 세트 | $1.99~$4.99 |
| 추가 이동 | +5 이동 | 프리미엄 재화 50 |
| 실패 시 이어하기 | 현재 상태에서 재개 | 프리미엄 재화 100 |
| 스킵 티켓 | 클리어한 스테이지 스킵 | 이벤트/상점 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 라이프, 부스터, 코인 |
| 인터스티셜 | 제한적 | 5스테이지마다 (스킵 가능) |
| 광고 제거 패스 | O | $2.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 스테이지 무과금 클리어 가능
- 부스터 없이도 3스타 달성 가능하도록 난이도 설계
- 꾸미기 아이템은 순수 외형 (기능 영향 없음)

### 과금 압박 최소화
- 캐주얼 유저 일일 플레이량은 라이프 자연 회복으로 충분
- 실패 시 바로 과금 유도하지 않음 (전략 힌트 제공)
- 광고는 완전 선택적
