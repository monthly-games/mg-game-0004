# MG-0004: Cafe Match UI/UX 스펙

> **게임명**: Cafe Match (카페 매치)
> **장르**: 매치-3 퍼즐
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 매치-3 표준 세로 그리드
- 스와이프 조작 최적화
- 상단 목표 + 하단 그리드 배치

## 2. Safe Area

| 영역 | 값 | 용도 |

### 3.1 상단 목표 영역

| 요소 | 크기 | 설명 |
|------|------|------|
| 레벨 표시 | 100×32dp | 현재 스테이지 |
| 수집 목표 | 3×(48×48dp) | 수집해야 할 아이템 |
| 이동 수 | 80×32dp | 남은 이동 횟수 |

### 3.2 매치 그리드

| 요소 | 크기 | 설명 |
|------|------|------|
| 셀 크기 | 72×72dp | 7×9 그리드 |
| 아이템 크기 | 64×64dp | 매치 아이템 |
| 특수 아이템 | 64×64dp | 폭탄, 줄무늬 등 |

### 3.3 부스터 바

| 요소 | 크기 | 설명 |
|------|------|------|
| 부스터 버튼 | 56×56dp | 4종 부스터 |
| 힌트 버튼 | 72×48dp | 힌트 표시 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 아이템 스와이프 | 셀 → 인접 셀 | 스와이프 (상하좌우) |
| 특수 아이템 활성화 | 아이템 위 | 탭 |
| 부스터 사용 | 56×56dp | 탭 → 타겟 선택 |
| 힌트 보기 | 72×48dp | 탭 |

### 터치 피드백
- 스와이프: 아이템 이동 애니메이션
- 매치 성공: 아이템 사라짐 + 파티클
- 콤보: 화면 쉐이크 + 스코어 팝업
- 실패 스와이프: 아이템 흔들림
