## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 스테이지 난이도 커브 조정 |
| 7~10주 | 카페 인테리어 테마 2~3종 추가 |
| 11~12주 | 첫 시즌 메뉴 이벤트 준비 |

**핵심 과제:**
- 퍼즐 난이도 밸런스 최적화
- 초반 이탈 방지
- 인테리어 시스템 안정화

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 단골 손님 이벤트 시작 |
| 5개월 | 시즌 메뉴 캠페인 1차 |
| 6개월 | 매치3 협동 이벤트 (서버 목표) |
| 7개월 | 시즌 메뉴 캠페인 2차 + 여름 테마 |
| 8개월 | 신규 인테리어 세트 대량 추가 |

**핵심 과제:**
- 단골 손님 시스템 효과 검증
- 협동 이벤트 참여율 향상
- 인테리어 다양성 확보

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 타이쿤 파트 강화 (매출/리뷰/평점 이벤트) |
| 10개월 | 다른 게임과 교차 이벤트 (카페 방문 보상) |
| 11개월 | 시즌 패스 도입 |
| 12개월 | 1주년 이벤트 + 연말 테마 |

**핵심 과제:**
- 타이쿤 경험 심화
- 크로스 게임 연동 검증
- 장기 유저 리텐션
