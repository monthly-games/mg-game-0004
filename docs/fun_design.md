## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 매치3 스테이지 플레이                                │
│         ↓                                               │
│  2. 카페 확장/인테리어/메뉴 개발                         │
│         ↓                                               │
│  3. 손님 응대/리뷰/평점 관리                             │
│         ↓                                               │
│  4. 이벤트/단골 손님과 상호작용 ─────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **매치3 스테이지**: 핵심 게임플레이, 퍼즐 클리어로 자원/스타 획득
2. **카페 확장**: 인테리어 배치, 메뉴 개발, 시설 업그레이드
3. **손님 응대**: 손님 요청 처리, 리뷰 평점 관리
4. **이벤트/단골**: 스토리 이벤트, 단골 손님 특별 상호작용

## 3. 재미 축 (Fun Pillars)

### 3.1 숙련/퍼즐 (Mastery)
- **목표**: 퍼즐 실력이 향상되는 느낌 제공
- **핵심 경험**: 어려운 스테이지를 실력으로 클리어했을 때 성취감
- **주요 요소**:
  - 콤보/체인 시스템
  - 특수 블록 생성
  - 제한 턴 내 목표 달성
  - 3스타 챌린지

### 3.2 꾸미기 (Expression)
- **목표**: 나만의 개성 있는 카페 만들기
- **핵심 경험**: 내 취향대로 꾸민 카페에 대한 애착
- **주요 요소**:
  - 인테리어 테마 선택
  - 가구 배치 자유도
  - 시즌 한정 데코
  - 손님 반응 피드백

### 3.3 스토리/캐릭터 애착 (Narrative Lite)
- **목표**: 가벼운 스토리로 감정적 연결 형성
- **핵심 경험**: 단골 손님들의 이야기가 궁금해지는 느낌
- **주요 요소**:
  - 단골 손님 스토리
  - 요일별 이벤트
  - 시즌 스토리
  - 캐릭터 성장 서사

## 4. 디자인 레버 (Design Levers)

### 4.1 카페 맥락 연결
| 레버 | 설명 |
|------|------|
| 블록/기믹/미션 연결 | 블록/기믹/미션을 카페 맥락(주문, 메뉴, 손님)과 강하게 연결 |
| 구현 방향 | 커피콩 블록 → 커피 제조, 케이크 블록 → 디저트 서빙, 손님 주문 = 미션 목표 |

### 4.2 인테리어 세트 효과
| 레버 | 설명 |
|------|------|
| 테마 맞추기 재미 | 인테리어 세트 효과 + 손님 반응 시스템 |
| 구현 방향 | 빈티지 테마 세트 → 문화인 손님 방문률 UP, 모던 테마 → 직장인 방문률 UP |

### 4.3 단골 손님 시스템
| 레버 | 설명 |
|------|------|
| 루틴 플레이 유도 | 단골 손님/요일 이벤트로 짧은 스토리와 루틴 플레이 유도 |
| 구현 방향 | 월요일 = 작가 지수씨 방문, 금요일 = 커플 데이트 이벤트 |

### 4.4 경영 지표 연동
| 레버 | 설명 |
|------|------|
| 평점 기반 콘텐츠 해금 | 카페 경영 지표(평점, 인기)에 따라 특수/하드 퍼즐 해금 |
| 구현 방향 | 평점 4.5 이상 → VIP룸 해금, 인기도 100 → 특별 주문 퍼즐 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 스테이지 클리어율 | 90% 이상 | 시작 → 클리어 전환율 |
| 일일 세션 수 | 5회 이상 | 접속 횟수 |
| 인테리어 변경률 | 주 3회 이상 | 배치 변경 이벤트 |
| 스토리 완료율 | 80% 이상 | 챕터별 완료 비율 |
| D7 리텐션 | 40% 이상 | 7일차 잔존율 |
